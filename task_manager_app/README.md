# Task Manager Application

## Project Overview:
Task Manager Application

## Features and Functionality:
Task Manager Application that helps users create to-do tasks, set it's time & date, edit and delete task. It also allows user mark as 'COMPLETED'.

## Technologies Used:
HTML, CSS & Javascript
